# Medical Bot With CI/CD

AI-powered healthcare follow-up assistant using:

- FastAPI
- LangChain
- Gemini
- GitHub Actions
- Pytest
- Ruff

## Features

- AI healthcare analysis
- ReAct agent architecture
- REST API
- Swagger documentation
- Automated testing
- CI/CD pipeline

## Run Locally

```bash
uvicorn app.main:app --reload
```
